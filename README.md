# CarritoApp

Aplicación móvil Flutter para ventas de ropa, con gestión de productos, pedidos y pagos (PayPal y contra entrega).
Incluye autenticación, administración, historial de pedidos y conexión con Firebase.

## Características principales
- Catálogo de productos y carrito de compras
- Registro e inicio de sesión de usuarios
- Pago con PayPal y contra entrega
- Panel de administración para productos, categorías y pedidos
- Historial de pedidos por usuario
- Integración con Firebase (Firestore y Storage)

## Instalación
1. Clona el repositorio:
   ```
   git clone https://github.com/tuusuario/CarritoApp.git
   ```
2. Instala dependencias:
   ```
   flutter pub get
   ```
3. Configura Firebase (agrega tus archivos de configuración).

## Uso
- Ejecuta en modo desarrollo:
  ```
  flutter run
  ```
- Compila APK:
  ```
  flutter build apk --release
  ```
